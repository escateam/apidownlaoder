# apidownlaoder
从api链接批量下载的工具
## 已实现功能
- [x] UI支持，直观简洁，更利于操作
- [x] 自定义下载，支持自定义下载链接，下载路径，下载次数，下载间隔
- [x] 双语言，支持Eghlish
- [x] 配置文件，在您每次开始下载的时候都会保存配置文件（下载链接还可以展开历史链接快速选择）
- [x] 一件跑路，删除该下载路径下的所有文件

> [!warning]
> 注意！注意！注意！一件跑路是删除该目录下的**所有文件**，并且*无法撤回*（后期可能会修复）
